MathQuill Fonts by Han and Jay

MathQuill currently uses the beautiful Unicode font Symbola by George Douros.

Note: mathquill.css includes from the font/ directory relative to its location, modify your version to include fonts from the right directory.

  • Symbola.eot, Symbola in Embedded OpenType, the only supported font format in Internet Explorer 6-7
  • Symbola.ttf, Symbola in TrueType, supported by all modern browsers
  • Symbola.otf, Symbola in OpenType, supported by all modern browsers, except for early versions of Chrome
  • Symbola.svg, Symbola in SVG, the only supported font format in Mobile Safari for iOS 4.1 and older
    • IIS must be configured it to deliver the correct MIME-type for SVG fonts
