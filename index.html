MathQuill

Typing Math Made Easy

Abstract

MathQuill is a formula editor for the web. It's an only slightly buggy, open source, JavaScript library.

Introduction

Here is a basic eaxmple with code for a MathQuill textbox and the equivalent LaTeX.

MathQuill:

LaTeX:

Code:

<p>MathQuill: <span id="math-field" style="min-width: 5em; min-height: 1.2em"></span></p>
<p id="latexp">LaTeX: <span id="latex"></span></p>

<script>
var MQ = MathQuill.getInterface(2); // guaranteed backcompat
var span = $('#math-field')[0];
var mathField = MQ.MathField(span, {
  spaceBehavesLikeTab: true, // configurable
  handlers: {
    edit: function() { // useful event handlers
      $('#latex').text(mathField.latex()); // simple API
    }
  }
});
</script>

About

MathQuill is written and maintained by @laughinghan, @jneen, and @stufflebear.

MathQuill has been helped by contributions and input from Desmos, whose slick graphing calculator makes extensive use of it.
