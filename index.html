MathQuill by laughinghan and jayferd

WYSIWYG math using only HTML and CSS

Try typing x^2:
Backspace and try 3/4, try \sin\theta, try \sqrt x.
Move around with the arrow keys, click, drag, cut, copy, paste.

MathQuill renders LaTeX math as semantically rich HTML (see our thorough demo) and presents it as human-readable math by styling with CSS. Showcasing the power of the HTML DOM, this math is WYSIWYG editable.

See also: our sister project Lolgebra, online chatrooms where you can “lol” in math!
